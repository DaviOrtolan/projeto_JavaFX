Este repositório foi criado para consolidar e organizar meus estudos em **JavaFX**. Aqui você encontrará uma coleção de exercícios práticos cobrindo desde os conceitos mais básicos de interface gráfica até a manipulação de layouts complexos e componentes de mídia.

---

## 🚀 Conceitos e Tópicos Praticados

O repositório está dividido em exemplos práticos que abrangem os seguintes pilares do JavaFX:

### 🏛️ Ciclo de Vida e Estrutura Básica
* **Application & Launch:** Entendimento do ciclo de vida de uma aplicação JavaFX e o método de inicialização.
* **Stage & Scene:** Manipulação do "palco" (janela principal) e da "cena" (conteúdo visual).
* **Start:** Implementação do método abstrato principal para renderização de telas.

### 🎨 Componentes e Layouts
* **Controls:** Uso de componentes interativos como `Button`, `TextField`, `Label`, `CheckBox`, entre outros.
* **Layouts:** Organização visual utilizando contêineres como `VBox`, `HBox`, `GridPane`, `BorderPane`, etc.
* **Listas:** Implementação e manipulação de `ListView` e `TableView` para exibição de dados estruturados.

### 🎬 Mídia e Elementos Visuais
* **Imagens:** Renderização de arquivos estáticos com `ImageView`.
* **Vídeos:** Reprodução de arquivos de mídia utilizando `Media` e `MediaPlayer`.

---

## 🛠️ Pré-requisitos e Tecnologias

Para rodar os projetos deste repositório, você precisará de:

* **JDK 17** ou superior.
* **JavaFX SDK** (configurado na sua IDE ou via gerenciador de dependências).
* Uma IDE de sua preferência (IntelliJ IDEA, Eclipse ou VS Code).

> 💡 **Dica:** Caso utilize o IntelliJ, lembre-se de adicionar os módulos do JavaFX nas configurações de execução (`--add-modules javafx.controls,javafx.media`).
